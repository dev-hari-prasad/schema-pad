We follow basic code of conduct guidelines to keep the community healthy and respectful. As this is a community-driven project, we expect all users to follow these guidelines. As the following:

- Be respectful and professional in all interactions
- Avoid using offensive or discriminatory language against any user or contributor regardless of their gender, race, religion, sexual orientation, disability, or any other characteristic
- **Do not** spam the issue tracker with multiple similar issues
- **Do not** spam the PRs with multiple similar PRs

It is common sense to follow these basic guidelines to keep the community healthy and respectful. If you are not sure about something, please ask the maintainers.
